# Repositioning Summary: De Política para Multi-Vertical

## 📊 O que Mudou

### Antes (V1)
```
Central de Monitoramento Político com IA
├─ Público: Campanhas, partidos, agências políticas
├─ Período: Sazonal (4 meses de eleição)
├─ Mercado TAM: ~R$32M
├─ Problema: Candidatos precisam monitorar sentimento político
└─ Ticket: R$2k-20k/mês
```

### Depois (V2 - Multi-Vertical)
```
SentimentHub / ReputationAI / PolarAI (a definir)
├─ Público: Políticos + Influencers + Brands
├─ Período: 365 dias/ano
├─ Mercado TAM: ~R$2B+
├─ Problema: Qualquer pessoa/marca precisa monitorar reputação digital
└─ Ticket: R$500-25k/mês (varia por vertical)
```

## 💰 Impacto Financeiro

### Receita Projetada (Ano 1)

**V1 (Só Política):**
```
Fase 1: 5 clientes × R$10k/mês = R$50k/mês (4 meses) = R$200k
Fase 2: (pausa de eleição por 4 anos)
Total Ano 1: ~R$200k
```

**V2 (Multi-Vertical):**
```
Ago-Set (eleição):
├─ Política: 5 clientes × R$10k = R$50k/mês
├─ Influencers: 5 clientes × R$15k = R$75k/mês
└─ Subtotal: R$125k/mês × 2 meses = R$250k

Out-Nov (consolidação):
├─ Política: 15 clientes × R$8k = R$120k/mês
├─ Influencers: 10 clientes × R$12k = R$120k/mês
└─ Subtotal: R$240k/mês × 2 meses = R$480k

Dez-Jul (pós-eleição):
├─ Influencers: 20 clientes × R$12k = R$240k/mês (cresce)
├─ Brands: 5 clientes × R$6k = R$30k/mês
└─ Subtotal: R$270k/mês × 8 meses = R$2.16M

Total Ano 1: ~R$2.9M (14x maior)
```

## ⚠️ Considerações

### Trade-offs
- **Antes:** Foco laser em 1 vertical
- **Depois:** Amplitude em 3 verticais
- **Mitigação:** MVP feature-complete para todas, depois otimizar por vertical

### Riscos
- **Complexidade:** Precisa entender 3 mercados diferentes
- **Mitigação:** Features core são agnósticas; variações são minor (UX copy)

### Oportunidades
- **White-label:** Agências de influencers revendem para seus clientes
- **Network effect:** Dados de múltiplas verticais = IA melhor
- **Expansion:** Depois pode ir para esportes, celebridades, políticos internacionais
