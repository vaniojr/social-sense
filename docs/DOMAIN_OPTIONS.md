# Opções de Domínios para SentimentHub

## 🎯 Resumo Executivo

Com o repositioning para **multi-vertical** (política + influencers + brands), precisamos de um nome que:
- ✅ Funcione para todas as verticais
- ✅ Comunique IA/inteligência
- ✅ Seja memorável e curto
- ✅ Tenha domínio disponível

---

## 🔍 Top 3 Recomendações

### 🥇 #1: SentimentHub

**Domínios a testar:**
- sentimenthub.com ⭐ (ideal)
- sentimenthub.io (fallback)
- sentimenthub.br (local)

**Por quê:**
- ✅ Direto: "sentimento" = core do produto
- ✅ "Hub" = centralizado, lugar de convergência
- ✅ Funciona para política, influencers, brands (todos precisam entender sentimento)
- ✅ Fácil de lembrar (2 palavras, 12 caracteres)
- ✅ Funciona em português e inglês
- ✅ Domínio simples (sem números ou hífens)

**Posicionamento:**
> "SentimentHub - A Central de Inteligência de Reputação. Monitore, entenda e responda ao que falam sobre você."

**Público:**
- Políticos: monitora sentimento do eleitor
- Influencers: monitora sentimento da audiência
- Brands: monitora sentimento de clientes

## 📋 Outras Opções (Alternativas)

### 4. EchoAI
- **Domínio:** echoai.com
- **Conceito:** "Eco" = repercussão, o que é dito sobre você
- **Vantagem:** Curto, único
- **Desvantagem:** Menos claro do que faz

### 5. PulseAI
- **Domínio:** pulseai.com
- **Conceito:** "Pulse" = tomar o pulso, medir temperatura
- **Vantagem:** Comunica real-time
- **Desvantagem:** Já existe PulseAI para saúde

### 6. ToneAI
- **Domínio:** toneai.com
- **Conceito:** "Tone" = tom da conversa, sentimento
- **Vantagem:** Curto, técnico
- **Desvantagem:** Menos descritivo

### 7. VoxAI
- **Domínio:** voxa.ai (premium)
- **Conceito:** "Vox" = voz, palavra pública
- **Vantagem:** Muito curto (3 caracteres)
- **Desvantagem:** Pode não ser claro para não-nativos de latim

### 8. AmpliBR / AmpliBR.ai
- **Domínio:** amplibr.com ou amplibr.ai
- **Conceito:** "Amplifi" + BR = amplifica voz brasileira
- **Vantagem:** Localizado (BR)
- **Desvantagem:** Limitado a mercado brasileiro

## ✅ Como Verificar Disponibilidade

### Opção 1: Namecheap (Recomendado)
```
1. Vá em https://www.namecheap.com/
2. Na barra de busca, digite: "sentimenthub"
3. Sistema mostra:
   - ✅ sentimenthub.com (disponível/preço)
   - ✅ sentimenthub.io (disponível/preço)
   - ✅ sentimenthub.br (disponível/preço)
4. Compare preços e escolha
```

### Opção 2: GoDaddy
```
1. Vá em https://www.godaddy.com/
2. Procure por domínio na home
3. Digite o nome que quer
4. Vê disponibilidade e preço
```

### Opção 3: Cloudflare Registrar
```
1. Vá em https://domains.cloudflare.com/
2. Busque domínio
3. Preços geralmente mais baratos que concorrentes
```

### Opção 4: Registro.br (para .com.br)
```
1. Vá em https://registro.br/
2. Busque domínio com terminação .com.br
3. Se disponível, registra direto (R$ 40/ano aproximadamente)
```
